This folder contains visual outputs from the Coffee Demand Forecasting Dashboard.

These images demonstrate key features of the application, including data analysis, visualization, and forecasting.

Dashboard Overview

This screenshot displays key performance metrics such as total revenue, total transactions, average order value, and peak hour.
It also shows overall trends to give a quick summary of business performance.


Demand Forecast

This screenshot shows future demand predictions using the Prophet forecasting model.
It includes a forecast line along with confidence intervals, helping visualize expected future trends.


Peak Hour Analysis

This screenshot represents a bar chart showing transaction distribution across different hours of the day.
It helps identify the busiest hours for the business.


Demand Heatmap

This screenshot shows a heatmap that visualizes transaction intensity across different stores and hours.
It helps compare demand patterns and identify high-activity periods.


Notes

All screenshots are taken from the Streamlit application
